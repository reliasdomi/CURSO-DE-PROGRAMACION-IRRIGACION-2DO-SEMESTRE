# Curso de Programacion Irrigacion - 2do Semestre

Repositorio publico con programas desarrollados para la materia de Programacion del grupo 4A del Departamento de Irrigacion.

## Autores

- Bistrain Borraz Angel Gabriel
- Cruz Sibaja Gibran Francisco
- Elias Dominguez Ruben
- Ortiz Ugarte Jonatan
- Torres Valencia Mario Alberto

## Contenido

- Programas en BASIC/QBasic/QB64.
- Menu principal y menus por categoria.
- Programas de figuras geometricas, estadistica, matrices y miscelaneos.
- Programas de normales climatologicas.
- Analisis, algoritmos y diagramas de flujo.
- Archivos de datos usados por los programas de climatologia.

## Nota

El repositorio incluye codigo fuente, documentos de apoyo, datos y diagramas de flujo. No se incluyen ejecutables compilados ni lanzadores locales.
